# Text Translator

Text Translator is a simple Python program that allows users to translate text into different languages using the Google Translate API.

## Features

- Translate text from one language to another.
- Display all available languages supported by Google Translate.
- Simple command-line interface.

## Requirements

- Python 3.x
- `googletrans` library (`pip install googletrans==4.0.0-rc1`)
- pip install --upgrade googletrans==4.0.0-rc1

## Usage

1. Clone the repository or download the `translator.py` file.
2. Install the required library using pip:
    ```
    pip install googletrans==4.0.0-rc1
    ```
3. Run the `translator.py` script:
    ```
    python translator.py
    ```
4. Follow the prompts to enter the text you want to translate and the language code you want to translate into.
5. To exit the program, enter 'q' when prompted for text input.
